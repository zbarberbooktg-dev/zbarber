Voici un prompt complet et bien structuré que tu peux coller dans Replit pour créer l’application Global Barber Corp :

Créer une application complète appelée Global Barber Corp, composée de trois parties principales :

1. Une application mobile iOS et Android pour les clients et les barbers.
2. Une plateforme web admin pour gérer l’ensemble du système.
3. Une base de données sécurisée pour centraliser les salons, services, réservations, abonnements et demandes de financement.
4. Un site web vitrine de l’appli mobile avec la description, les objectifs de l’appli, les liens de téléchargement 

Objectif général

Global Barber Corp est une plateforme destinée à centraliser les barbers en Afrique afin de donner plus de visibilité à leurs salons, faciliter la prise de rendez-vous pour les clients et offrir aux barbers des outils professionnels pour développer leur activité.

L’application doit permettre aux utilisateurs de trouver facilement les salons de coiffure les plus proches, consulter les services proposés, voir les avis, réserver un service et partager les profils de salons.

Pour les barbers, la plateforme doit leur permettre de créer un profil professionnel, gérer leurs salons, ajouter leurs services, afficher leur localisation, gérer leurs réservations, souscrire à un abonnement et éventuellement faire une demande de financement pour améliorer leur salon, leurs outils ou leurs services.

Rôles utilisateurs

1. Client / utilisateur simple

L’utilisateur peut :

* consulter les salons sans créer de compte
* voir les services proposés par chaque salon
* rechercher un salon par nom, ville, quartier, prix ou service
* voir les salons proches grâce à la géolocalisation
* créer un compte
* se connecter
* réserver un service
* mettre un salon ou barber en favori
* laisser un avis et une note
* partager le lien d’un salon
* voir l’historique de ses réservations
* recevoir des notifications de rappel de rendez-vous

La création de compte n’est pas obligatoire pour consulter les salons et services, mais elle est obligatoire pour réserver, mettre en favori, noter ou laisser un avis.

2. Barber / salon

Le barber peut :

* créer un compte professionnel
* créer un profil barber
* ajouter le logo ou la photo de son salon
* ajouter les informations du salon : nom, adresse, ville, quartier, téléphone, WhatsApp, horaires, description
* ajouter une localisation précise sur la carte
* créer, modifier et supprimer ses services
* ajouter les prix, durée, description et photos des services
* gérer les réservations reçues
* accepter, refuser ou confirmer un rendez-vous
* modifier son profil
* souscrire à un abonnement
* voir son statut d’abonnement
* faire une demande de financement pour optimiser son salon
* voir les conférences ou annonces publiées par l’administration
* recevoir des notifications importantes

3. Administrateur

La plateforme web admin doit permettre de :

* valider ou refuser les comptes barbers
* gérer tous les comptes clients
* gérer tous les barbers et salons
* suivre les services proposés par les salons
* gérer les réservations
* gérer les abonnements barbers
* gérer les barbers éligibles au programme de financement
* valider, refuser ou suivre les demandes de financement
* publier des annonces et conférences pour les barbers
* gérer les catégories de services
* gérer les avis et signalements
* voir les statistiques globales
* gérer les contenus affichés dans l’application
* suspendre ou supprimer un compte si nécessaire
